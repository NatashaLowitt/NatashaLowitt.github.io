Drop

Drop is a web application designed to raise awareness about water consumption specifically in California where they are currently experiencing a severe drought. The two main features are a consumption calculator and an explore feature. The consumption calculator allows the user to input the number of times they did the laundry, washed their hands, showered, etc. Then it tells the user the number of gallons they consumed of water and some tips to limit water consumption. The explore feature visually compares the gallons of water per capita daily used by the main cities of California. Then, there are special profile pages for each city individually with an interactive map, thanks to the google maps API.

— Made in Advanced Software Engineering at Flatiron School —

Friendly Message Sender

A web application that allows the user to input the name, phone number, and category of the desired compliment. The web app generates a random compliment from the desired category and sends it as a text message to the desired recipient. According to a study on Positive Psychology, even one compliment can make a major impact on someone's day.
